ProbeBasicBebasMono.ttf in this repository is a modified version of the upstream Bebas Kai font.

Original font:
- Name: Bebas Kai
- Author: Ryoichi Tsunekawa (Dharma Type)
- Copyright: Copyright (c) 2014 by Dharma Type.
- License: SIL Open Font License 1.1 (see OFL-1.1.txt)

Repository-local modifications:
- Numeric glyph spacing was adjusted to fixed-width (tabular/monospaced digits)
  so DRO/G-code numeric fields do not shift horizontally while values update.
- These modifications were made in this project history in 2019.
- The distributed filename in this project is `ProbeBasicBebasMono.ttf`.
- Internal font naming metadata has been updated to the family name
  `Probe Basic Bebas Mono`.

Distribution note:
- This font remains distributed under SIL Open Font License 1.1.

  redistributing this project or binary packages.
